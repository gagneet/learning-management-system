# **5. Class Scheduling**

## **5.1 Creating a Class**

1. Navigate to **Classes ‚Üí Create Class**  
2. Select tutor  
3. Add students  
4. Choose schedule (one‚Äëoff or recurring)  
5. Select video provider (Teams/Zoom/Chime)

## **5.2 Managing Class Changes**

### **Rescheduling**
- Tutors or parents may request changes

- Tutors may cancel

- System suggests alternative tutors
